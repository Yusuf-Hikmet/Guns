# Guns

A Roblox Studio gun system featuring multiple weapons.

## Weapons
- M4A1
- Remington 870
- Remington 700
- Deagle

## Features
- Server-side damage handling
- ViewModel animations
- Inventory system
- DataStore
- Shop system
- Game loop
